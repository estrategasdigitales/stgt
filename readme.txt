1.- El proyecto viene en PHP únicamente para colocar los include de header y footer en cada página, fuera de eso, todo es estructura HTML, CSS, BOOTSTRAP

2.- El CSS lo encontraran dividido por bloques, donde cada bloque hace alusión a que se le aplica estilos, por ejemplo, /*header*/ /*footer*/ etc

3.- Al final del CSS vienen 2 media query, uno para tablet y el otro para mobile

4.- En los HTML encontraran comentarios de cada bloque

5.- Los modales, carruseles, tabs son empleados de boostrap
https://getbootstrap.com/docs/5.2/getting-started/introduction/

6.- Se incluye la librería de FontAwesome para algunos iconos

7.- Se incluye la fuente de Google Fonts

8.- En la carpeta img vienen las imágenes utilizadas

9.- En el caso de la maqueta de viajes, considerar que algunos componentes tiene ID's que deben ir cambiando según el número de corridas
